One-time setup

  1. Go to console.cloud.google.com → create a project
  2. Enable Gmail API and Google Calendar API in Library
  3. OAuth consent screen → External → add your email as test user
  4. Credentials → Create OAuth Client ID → Web application
    Add to Authorized JavaScript origins
  5. Paste your Client ID below and click Connect

Your Client ID is stored in localStorage. Tokens are stored in sessionStorage and cleared when you close the tab.
